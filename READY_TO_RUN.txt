DATABASE MANAGEMENT
════════════════════════════════════════════════════════════════════════════════

█ VIEW DATABASE CONTENTS:
  Download "DB Browser for SQLite" (free):
  https://sqlitebrowser.org/
  
  Then: File → Open → Select SmartPOS.db

█ RESET DATABASE:
  Just delete SmartPOS.db file
  Next run will recreate it with fresh data

█ BACKUP DATABASE:
  Copy SmartPOS.db to another location
  Example: Copy-Item SmartPOS.db SmartPOS_backup.db

█ MOVE DATABASE:
  Edit appsettings.json:
  "DefaultConnection": "Data Source=C:\\MyFolder\\SmartPOS.db"


TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

Problem: Application doesn't start
Solution: 
  1. Check .NET 8 is installed: dotnet --version
  2. Rebuild: dotnet build
  3. Try again

Problem: Login window doesn't appear
Solution:
  1. Check console for errors
  2. Look for error messages
  3. Check SmartPOS.db was created
  4. Try pressing Alt+Tab (window might be behind others)

Problem: "Database is locked"
Solution:
  1. Close all instances of the application
  2. Close DB Browser if open
  3. Try again

Problem: Invalid credentials
Solution:
  1. Use exactly: admin / admin123 (lowercase)
  2. Or use quick access buttons
  3. Check console for "Default users created" message
