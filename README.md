# Smart Retail POS Management System

A comprehensive, AI-powered desktop Point of Sale (POS) system built with C# and WPF on .NET 10.

## 📖 Documentation
All system documentation has been organized into the `docs/` folder:

- **[Project Overview](docs/PROJECT_OVERVIEW.md)**: System goals, architectural design, and project structure.
- **[Development Roadmap](docs/PHASE_ROADMAP.md)**: Multi-phase status and upcoming milestones.
- **[Developer Handbook](docs/DEVELOPMENT_GUIDE.md)**: Setup guides, instructions on database migrations, and build commands.
- **[Database Schema](docs/DATABASE_SCHEMA.md)**: Detailed tables and column definitions.
- **[Service API Reference](docs/API_DOCUMENTATION.md)**: Backend services and interface definitions.
- **[User Guide](docs/USER_GUIDE.md)**: Instructions for default test credentials, initial boot, and usage instructions.
- **[Development Log](docs/DEVELOPMENT_LOG.md)**: Timeline of milestones and phase completions.

## 🚀 Quick Start
To build and run the application locally:

```powershell
# Restore dependencies and build the solution
dotnet build

# Run the UI presentation layer
dotnet run --project SmartPOS.UI
```

Refer to the **[Developer Handbook](docs/DEVELOPMENT_GUIDE.md)** for detailed database migration and connection configurations.
