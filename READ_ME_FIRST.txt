🆘 TROUBLESHOOTING:

"Command doesn't respond"
→ Close Git Bash and use PowerShell instead

"Window doesn't appear"
→ Check if it's behind other windows (Alt+Tab)

"Error in console"
→ Read the error message carefully
→ Check logs/smartpos-*.txt for details

"Connection failed"
→ SQL Server LocalDB might not be installed
→ Or connection string in appsettings.json is wrong

"File not found"
→ Make sure you're in: D:\Smart-POS\Smart-POS
→ Run: cd D:\Smart-POS\Smart-POS first
