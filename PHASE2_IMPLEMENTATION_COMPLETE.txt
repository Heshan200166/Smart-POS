TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

Problem: Login fails
Solution: Check username (admin) and password (admin123)
          Ensure user is Active (not deactivated)

Problem: Access denied to user management
Solution: Only Admin can manage users
          Login as admin user instead

Problem: Application won't start
Solution: Check .NET 8 is installed
          Verify SQL Server LocalDB is available
          Check database connection string in appsettings.json

Problem: Window appears behind other windows
Solution: Press Alt+Tab to switch windows
          Or click taskbar icon
